
데이터 타입

데이터 타입은 기본형과 참조형으로 구분 된다.

- 기본형 (primitive type)
    - Number
    - String
    - Boolean
    - null
    - undefined
    - Symbol (es6 추가 됨)

- 참조형 (Reference type) - object
    - Array
    - Funtion
    - RegExp
    - Map, WeakMap
    - Set, WeakSet

기본형과 참조형을 구분 하는 차이는

기본형은 값이 담긴 주솟값을 바로 복제

참조형은 값이 담긴 주소값들로 이루어진 묶음을 가리키는 주솟값을 복제
